### Maychew Martyrs Memorial School Management System

This is a web application being built to manage the students, teachers,
and academic operations at MMM Special Boarding School.

🔑 Live Demo
https://mmm-boarding-high-school-managing-system.onrender.com/

To review the system, use these credentials:

  - Email: admin@MMM.com
  - Password: password

🚀 Main Features

  - Dashboard: View total students, teachers, and gender ratios.
  - Student Management: Add, edit, and view student profiles.
  - Teacher Management: Manage faculty records and course assignments.
  - Academic Setup: Manage Classes, Sections, and Semesters.
  - Attendance: Take and track daily student attendance.
  - Exams & Grading: Set grading rules and calculate final marks.
  - Notice Board: Post school-wide announcements.

🛠 Built With

  - PHP 8.3 (Logic)
  - Laravel 8 (Framework)
  - PostgreSQL (Database)
  - Bootstrap 5 (Design)
  - Docker (Deployment)

💻 How to Run Locally

1.  Clone the code: git clone
    https://github.com/tedd12t/MMM-Boarding-High-School-Managing-System.git
2.  Install: composer install
3.  Setup Database: Update your .env file and run php artisan migrate --seed
4.  Launch: php artisan serve
